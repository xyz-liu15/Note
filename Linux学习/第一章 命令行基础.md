## 3. cd

**绝对路径**

*定义：*从**根目录（/ 或 C:\）**开始的完整路径，指向文件或目录的**精确位置**。

**相对路径**

**定义：**基于**当前目录**的路径，指向目标文件或目录的**相对位置**。

绝对路径与相对路径对比：

| **特点**   | **绝对路径**         | **相对路径**         |     |
| -------- | ---------------- | ---------------- | --- |
| **起点**   | 根目录（`/` 或 `C:\`） | 当前目录（`.` 或 `..`） |     |
| **稳定性**  | 始终有效             | 依赖当前目录位置         |     |
| **适用场景** | 脚本、跨目录引用文件       | 项目内部文件引用         |     |

一直使用绝对路径和相对路径导航会相当累人，幸运的是有一些快捷方式可以帮助你。

- .（当前目录）。这是你当前所在的目录。
- ..（父目录）。带你到当前目录的上一级目录。
- ~（主目录）。这个目录默认为“主目录”。
- -（上一个目录）。这将带你到刚才所在的上一级目录。
## 4. ls
默认情况下，***ls*** 命令将列出当前目录中的目录和文件，但是您可以指定要列出目录的路径。

![[ls.png]]

需要注意的是，当文件夹中存在以 **.** 开头的文件或者文件夹时，运行 ***ls*** 不会显示这些文件或者文件夹，这时要想显示它们时，需要运行 ***ls -a*** 。

还有一个非常有用的 ***ls*** 标志，即 ***-l*** 标志用于长格式，它会以长格式显示文件的详细列表。这将显示详细信息，从左边开始： ***文件权限、链接数量、所有者名称、所属组、文件大小、最后修改时间戳以及文件/目录名称*** 。

如果要查看详细信息时，可以运行 ***ls -al*** 或者 ***ls -la*** 。

![[ls _plus.png]]

## 9. history

另一个历史快捷键是 ***ctrl-R*** ，这是反向搜索命令，如果你按下 ***ctrl-R*** 并开始输入你想要的命令的部分，它会显示匹配项，你可以通过再次按 ***ctrl-R*** 键来浏览它们。一旦你找到了你想再次使用的命令，只需按 ***Enter*** 键即可。

## 10. cp

让我们开始复制这些文件。就像在其他操作系统中复制粘贴文件那样，shell 给我们提供了一种更简单的方法。

```Shell
cp mycoolfile /home/pete/Documents/cooldocs
```

mycoolfile 是你想复制的文件，/home/pete/Documents/cooldocs 是你要将文件复制到的位置。

你可以复制多个文件和目录，同时也能使用通配符。通配符是一种可以替代特定模式的字符，使搜索更加灵活。你可以在任何命令中使用通配符，从而提升操作的灵活性。

- * 通配符中的万能通配符，用于表示所有单个字符或任何字符串。
- ? 用来表示一个字符
- [] 用于表示括号内的任何字符

```Shell
cp *.jpg /home/pete/Pictures
```
这将会把当前目录下所有 .jpg 格式的图片文件复制到 Pictures 目录。

一个有用的命令是使用 -r 选项，这将递归复制目录内的文件和子目录。

尝试将一个包含若干文件的目录复制到你的 Documents 目录。没成功？这是因为你需要使用 -r 命令来同时复制里面的文件和子目录。

```Shell
cp -r Pumpkin/ /home/pete/Documents
```

有一个需要注意的地方，如果你将文件复制到一个包含相同文件名的目录中，该文件将被你正在复制的内容覆盖。如果你有不想被意外覆盖的文件，这可不是什么好消息。你可以使用 -i 标志（交互式模式）来在覆盖文件之前进行提示。

```Shell
cp -i mycoolfile /home/pete/Pictures
```

## 11. mv

用于移动文件并重命名它们。在标志和功能方面与cp命令非常相似。

你可以像这样重命名文件：

```Shell
mv file_1 file_2
```

或者你可以将文件移动到不同的目录：

```Shell
mv file /home/Document
```

也可以移动多个文件：

```Shell
mv file_1 file_2 /home/Document
```

你也可以重命名目录：

```Shell
mv directory_1 directory_2
```

就像 ***cp*** 一样，如果你使用 ***mv*** 移动一个文件或目录，它将会覆盖同一目录下的任何内容。因此，你可以在覆盖任何内容之前使用 ***-i*** 标志来提示你。

```Shell
mv -i directory_1 directory_2
```

假设你想将文件移动以覆盖之前的版本。你也可以备份该文件，它将只是将旧版本重命名为一个 ~。

```Shell
mv -b directory_1 directory_2
```

## 12. mkdir

我们需要一些目录来存储我们一直在工作的所有文件。***mkdir*** 命令（创建目录）对此很有用，如果目录不存在，它将创建目录。你甚至可以一次创建多个目录。

```Shell
mkdir books paintings
```

您也可以同时使用 ***-p*** （父目录标志）创建子目录。

```Shell
mkdir -p books/novels/xuanhuan
```

## 13. rm

***rm*** 命令用于删除文件和目录。

```Shell
rm file
```

**注意：** 使用 ***rm*** 时要小心，没有垃圾桶可以找回已删除的文件。一旦删除，就永远消失了。
幸运的是，有一些安全措施已经实施，这样普通人不会轻易删除重要文件。受写保护的文件在删除前会提示确认。如果一个目录受写保护，它也不会被轻易删除。

现在如果你不在乎这些，完全可以删除大量文件。

```Shell
rm -f file
```

***-f*** 或者 ***force*** 选项告诉 ***rm*** 命令删除所有的文件，无论是否受到保护，而无需提示用户（只要你有适当的权限）。

```Shell
rm -i file
```

像许多其他命令一样，添加 ***-i*** 标志会提示你确认是否真的要删除文件或者目录。

```Shell
rm -r directory
```

默认情况下你不能直接使用 ***rm*** 删除目录，你需要递归的删除（添加 ***-i*** ）所有文件及其可能存在的子目录。

你可以使用 ***rmdir*** 删除目录。

```Shell
rmdir directory
```

## 14. find

系统上有这么多文件，要找到特定的一个可能会有些混乱。不过，我们可以使用一个命令来解决这个问题，那就是 ***find*** 命令！

```Shell
find /home -name puppies.jpg
```

使用find命令时，需要指定要搜索的目录，要查找的内容.

还可以指定您要查找的文件类型。

```Shell
find /home -type d -name MyFolder
```

你可以看到，我已经将我想要查找的文件类型设置为(d)代表目录，并且我仍在通过MyFolder的名称进行搜索。

值得注意的是，***find*** 命令不会停止在你要搜索的目录，它还会查看该目录下可能存在的任何子目录。

## 15. help

Linux有一些很棒的内置工具，可以帮助您了解如何使用命令或检查命令可用的标志。其中一个工具，即 ***help*** ，是一个内置的bash命令，它为其他bash命令（如echo、logout、pwd等）提供帮助。

```Shell
help echo
```

这将为你提供一个描述，以及当你想要运行echo时可以使用的选择。对于其他可执行程序，通常有一个名为--help或类似的选择。

```Shell
echo --help
```

并非所有发布可执行文件的开发者都会遵守这个标准，但可能这是你在程序上找到帮助的最佳机会。

## 16. man

哎呀，我希望这些程序中的一些有手册，这样我们就可以看到更多关于它们的信息。幸运的是，它们确实有！恰如其名的 ***man*** 页面，你可以使用 ***man*** 命令查看命令的手册。

```Shell
man ls
```

Man手册是默认集成在大多数Linux操作系统中的手册。它们提供了关于命令和系统其他方面的文档。
尝试在几个命令上试用，以获取更多关于它们的信息。

## 17. whatis

唉，到目前为止我们已经学了很多命令，如果你对某个命令的作用感到怀疑，可以使用      ***whatis*** 命令。***whatis*** 命令提供了命令行程序的简要描述。

```Shell
whatis cat
```

描述来源于每个命令的手册页面。如果你运行了whatis cat，你会看到一个带有简短描述的小段落。

## 18. alias

有时输入命令可能会变得非常重复，或者如果你需要多次输入一个长的命令，最好有一个你可以使用的别名。要为命令创建别名，你只需指定一个别名名称并将其设置为命令。

```Shell
alias foobar='ls -la'
```

现在你不必再输入 ***ls -la*** ，你可以输入 ***foobar*** ，它就会执行那个命令，这真是个很酷的功能。请 **注意** ，这个命令在重启后不会保存你的别名，所以你需要在以下位置添加一个永久别名：

```Shell
~/.bashrc
```

或者类似的文件，如果您希望在重启后保持其存在。

您可以使用 ***unalias*** 命令来删除别名：

```Shell
unalias foobar
```
